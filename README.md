SPINKIFY adalah aplikasi pemutar musik desktop yang dikembangkan sebagai proyek Tugas Besar Mata Kuliah Struktur Data. Aplikasi ini dibangun menggunakan Python dan Tkinter dengan tema antarmuka dark charcoal dengan aksen pink untuk memberikan pengalaman visual yang modern dan nyaman digunakan.

Aplikasi ini mengimplementasikan struktur data berikut:

1. **Doubly Linked List (DLL)**  
   Digunakan untuk menyimpan library lagu. Struktur ini dipilih karena mendukung navigasi dua arah (next dan previous) yang dibutuhkan dalam proses pemutaran lagu.

2. **Singly Linked List (SLL)**  
   Digunakan untuk mengelola playlist lagu. Struktur ini sesuai karena playlist bersifat linier dan tidak memerlukan navigasi dua arah.

3. **Record (Tipe Data Bentukan)**  
   Setiap lagu direpresentasikan dalam bentuk record yang berisi atribut seperti ID lagu, judul, artis, genre, album, dan tahun rilis.

SPINKIFY menyediakan dua peran utama, yaitu Admin dan User. Admin berfokus pada pengelolaan data lagu, sedangkan User dapat memutar lagu, mengelola playlist, serta menggunakan fitur navigasi pemutaran.
